# Network Port Scanner GUI

## Overview
The Network Port Scanner GUI application is designed to provide users with a graphical interface to scan and analyze open ports on networked devices. It enables users to easily identify which ports are open, filtered, or closed, making it a valuable tool for network administrators and security professionals.

## Features
- **User-Friendly Interface:** An intuitive GUI that makes it easy to perform port scans without needing to use command-line tools.
- **Port Scanning:** Ability to scan for common ports as well as custom ports specified by the user.
- **Result Display:** Clearly displays the status of each port (open, closed, or filtered) in real-time.
- **Export Results:** Option to export the scanning results to a CSV or text file for further analysis.

## Installation
To install the Network Port Scanner GUI, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/Rishikesh15452/Network-Port-Scanner-GUI.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Network-Port-Scanner-GUI
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the application:
   ```bash
   python main.py
   ```

## Usage
- Launch the application and specify the target IP address.
- Choose the range of ports to scan (or use the default settings).
- Click the 'Scan' button to start the scanning process.
- Review the results displayed in the GUI.
